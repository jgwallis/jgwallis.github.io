Line Noise

This test determines whether there is any noise on the telephone line. By the way, if there is noise on the line then it counts as a telephone service fault rather than a broadband fault, and will need to be reported as such.

  1. Connect a wired telephone to the test socket.
  2. Call BT’s line test facility on 17070.
  3. Choose the ‘quiet line test’ option from the menu.
  4. Press the mute button to avoid hearing background noise from your own location.
  5. Listen for line noise.
  6. Repeat the test at different times and in different conditions. NOTE: If you try the test without the router attached and there is no noise, then you repeat the test with the router attached and there is noise, the router itself is creating that noise.
