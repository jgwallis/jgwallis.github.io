Router IP Addresses

If you do not know the IP address of the router, there is a simple Windows procedure to find it.

  1. Open a command window (run ‘cmd.exe’).
  2. Type ‘ipconfig’ and press the return key.
  3. Look under ‘Ethernet adapter Local Area Connection’ or ‘Wireless LAN adapter Wireless Network Connection’ for the ‘Default Gateway Address’, which is the router’s IP address.
  4. Close the command window by typing ‘exit’ and pressing the return key.
