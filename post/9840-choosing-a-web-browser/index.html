Choosing a Web Browser

Always install several web browsers

It is always a good idea to install several web browsers on your computer in case the one that you normally use starts causing problems. The latest version of Windows, which is Windows 10, has a built-in web browser called Microsoft Edge. Earlier versions of Windows had a built-in web browser called Microsoft Internet Explorer. These browsers have been so deeply integrated with Windows that it has always been practically impossible to get rid of them, but there are plenty of other web browsers that you can install and use instead.

Picking a favourite

The two most popular alternative web browsers are Google Chrome and Mozilla Firefox. There are other alternative web browsers too, such as Opera and Safari, but they are installed relatively rarely. Mozilla Firefox is produced by a not-for-profit foundation, so it is not commercially orientated, but Google Chrome has a few advantages:

  • Google Chrome can save most web pages for offline viewing, whereas for some reason Mozilla Firefox often fails to save the very same pages without giving any clues as to why it has failed
  • Google Chrome can search the web based on voice commands, which can save a lot of typing, whereas Mozilla Firefox unfortunately can’t
  • For developing new websites, Google Chrome provides what I consider to be more comprehensive debugging facilities
