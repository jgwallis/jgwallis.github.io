How the site was built

There’s no complicated content management system behind this site, so it should load quickly and be very secure. It was built ‘statically’ using the Hugo static site generator, which is available under version 2.0 of the Apache Licence, and the UIkit 3 framework from YOOtheme, which is available under the MIT licence.

You probably don’t care about any of these technicalities, but if you’d like help creating a site of your own then please feel free to get in touch (this site has deliberately been kept simple, but it is possible to use the same tools to produce something much more sophisticated).
