How the site was built

There’s no complicated content management system behind this site, so it should load quickly and be very secure. It was built ‘statically’ using Sassc, Bourbon, Neat, Bitters, Normalize, and Hugo. You probably don’t care about any of these technicalities, but if you’d like help creating a site of your own then please feel free to get in touch.

Software used to build the site

The versions of the various different bits of software used to build this site are:
