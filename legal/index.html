How the site was built

There’s no complicated content management system behind this site, so it loads quickly and is very secure. It was built ‘statically’ using the Hugo static site generator (© The Hugo Authors), which is available under version 2.0 of the Apache Licence, and UIkit (© YOOtheme GmbH), which is available under the MIT licence.

You probably don’t care about any of these technicalities, but if you’d like help creating a site of your own then please feel free to get in touch (this site has deliberately been kept simple, but it is possible to use the same tools to produce something much more sophisticated).

Software used to build the site

The different software items used to build this site were:

  • Hugo version 0.83 (Hugo is probably the fastest of all static site generators)
  • UIkit version 3.6.21 (UIkit looks good and doesn’t depend on lots of other software)

Compatibility

This site displays properly in all modern browsers, but may not look quite as intended in ancient versions of Microsoft’s Internet Explorer, which were never very good at sticking to standards even when they were current.
