Setup

Setting up new systems, refreshing existing systems, and decommissioning old systems:

  • Removing old hardware and cabling then installing new equipment
  • Transferring data from old systems to new ones (programs cannot be easily copied across, so they usually have to be re-installed)
  • Proper disposal of unwanted equipment (you can’t just throw it in the bin because of the Waste Electrical and Electronic Equipment Regulations 2013)
  • Helping to make the right decisions when using Windows for the first time (such as whether or not to let Microsoft keep a close eye on your activities)
  • Securely wiping and physically destroying hard disk drives (which can still contain sensitive data even after you think you’ve obliterated it all)
  • Putting the start button back into Windows 8 and 8.1, and making Windows 10 menus behave like more familiar menu systems
  • Reinstalling operating systems (back to ‘factory settings’)
  • Replacing one operating system with another (switching from Windows to Linux will often breathe new life into an old system)
  • Installing operating systems side-by-side (you can easily have Windows and Linux installed on the same hardware as a ‘dual-boot’ system)
  • Destroying all data on an old system prior to sale or donation
  • Installing a new router after a change of Internet service provider
